# 📝 StickerApp

**Миниатюрные стикеры-заметки, которые живут прямо на рабочем столе.**

*(WPF · .NET 9 · Windows)*

---

## ✨ Что это

StickerApp — это не просто "ещё одна программа для заметок". Это маленькие живые листочки, которые:

- 🍃 **качаются** на ветру, даже когда ты просто смотришь на них
- 🖱 **пружинят и крутятся**, когда их таскаешь за заголовок
- 🎨 **меняют цвет** по настроению
- 🏀 **отскакивают от стенок** — да, тут есть резиновый мячик со звуком *boing*

Всё сохраняется автоматически. Перезагрузил компьютер — стикеры на месте.

---

## 🎬 Возможности

| Фича | Описание |
|------|----------|
| 🖊 **Быстрые заметки** | Нажал `Ctrl + Alt + Mouse4` — стикер появился под курсором. Начал печатать сразу. |
| 🎨 **Цвета настроения** | Каждый стикер имеет свой акцентный цвет. Меняется в один клик. |
| ✔ **Чек-листы** | Внутри стикера — список задач. Галочка, дело сделано. |
| 🖱 **Физика перетаскивания** | Тянешь за заголовок — стикер чуть приподнимается и отпружинивает обратно. |
| 🏀 **Резиновый мяч** | Бросаешь — он летит, отскакивает от краёв экрана со звуком. Безумно satisfying. |
| 🔔 **Иконка в трее** | Тихо сидит рядом с часами, не мешает. |
| 💾 **Автосохранение** | Всё пишется в `%APPDATA%`. Забудь про Ctrl+S. |

---

## ⌨ Горячие клавиши

| Комбинация | Действие |
|-----------|----------|
| `Ctrl + Alt + Mouse4` *(боковая кнопка мыши)* | Создать стикер под курсором |
| `Enter` | Добавить задачу в активный стикер |
| `Escape` | Отменить ввод |
| `Backspace` | Стереть символ |

> 💡 **Pro-tip:** стикеры никогда не отнимают фокус у других окон — можно печатать в коде и тут же добавлять задачу, не переключаясь.

## 🚀 Установка

### Требования
- Windows 10/11
- [.NET 9 Runtime](https://dotnet.microsoft.com/download/dotnet/9.0)

### Сборка из исходников

```bash
git clone https://github.com/username/StickerApp.git
cd StickerApp
dotnet build Notes.sln
```

Или запусти `build.bat`, если ты на Windows.

---

## ⚙ Настройки

Все настройки хранятся в `%APPDATA%\StickerApp\settings.json`:

```json
{
  "DefaultAccentColor": "#FF5A9"
}
```

Цвет по умолчанию — кораллово-розовый, но ты можешь выбрать любой через окно настроек.

## 🏗 Архитектура

```
StickerApp/
├── App.xaml.cs              → Трей, хуки, жизненный цикл
├── StickerWindow.xaml/.cs   → Окно стикера, анимации, физика
├── StickerData.cs           → Модель: ID, заголовок, цвет, задачи
├── StickerStore.cs          → JSON-персистентность
├── AppSettings.cs           → Глобальные настройки
├── RubberBallWindow.xaml.cs → 🏀 Мячик с boing-звуком
└── Win32.cs                 → P/Invoke для низкоуровневой магии
```

**Стек:** .NET 9 WPF + WinForms (трей-иконка), C# 12, `System.Text.Json`. Никаких внешних зависимостей.
